# springdoc

Spring Boot REST API documentation with Swagger.

## Installation

Install the app on terminal

```
git clone git@github.com:teguharifudin/springdoc.git
```
```
cd springdoc
```

## Usage

Running the app on http://localhost:8080/swagger-ui/index.html
